POTEPAN CAMPでは複数のコースをご用意しており、
受講頂く方のこれまでのご経歴を基に最適なコースをご案内しております。

実務経験が無いが、今後エンジニアとしてのキャリアを形成していきたい方

実務経験はあるがブランクが長く、まずは最新の技術動向も思い出したい方

独学で勉強しているが、今一自身のスキルに自信を持つ事が出来ない方

様々な背景を持った方をこれまで受け入れてきた実績があります。
実際のコース決定は社内のカウンセリング担当と行いますのでご安心ください。

  • 受講内容

    実践的な課題開発を3ヶ月間行なって頂きます。

    HTML CSS JavaScript Ruby Ruby on Rails Rspec Docker heroku rubocop PostgreSQL Solidus Git Bitbucket Slack

  • コース概要

  • 受講内容

    Webアプリ開発に必要な要素を全体的に学び、実践的な開発まで行います。
    分かりやすい教材を用いているのでオンラインでも非常に学びやすくなっております。
    最後は実際にWebアプリを開発しレビュアーにチェックしてもらいます。

  • 受講期間

    2ヶ月（9週間）

  • 応募条件

    MacBookを持っている方

  • 受講料金

    HTML CSS JavaScript Ruby Ruby on Rails

  • 課題フロー

  • 01基礎学習

    分かりやすい教材を用いてWeb開発の基礎を学習します。 HTML,CSS,JavaScript,Ruby,Ruby on Rails Git コマンドラインを触るところから始まります。 特にマークアップやRubyのスキルを高めていきます。
